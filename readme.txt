Convert Media Library images in next-gen format and speed up your website

== Description ==

Speed up your website by serving images in WebP format. This plugin will replace files in standard JPEG, PNG and GIF formats with WebP format.

After installation go to plugin settings and enable webp images. You can convert all images in Media Library or convert selected images only. The plugin will reduce the size of the image without compromising image quality. 

The plugin will ditect if Webp images are supported by the client browser or not and then serve the images accordingly.

== Installation ==

1. Upload `webp-converter.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Click on Webp Convert and start converting images.
